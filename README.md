# Drawy
An embedded project that displays a drawing made on a program on LED matrices

## Функционалности
Drawy предоставя удобна за използване платформа за рисуване. В нея може да се рисува, да се подбира цвят чрез color wheel, както и да се наглася brightness. През програмата могат и да се запазват и зареждат изображенията.

## Технологии

|----------------|-----------|
